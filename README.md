# 🛒 Whatbytes E-Commerce Platform

A premium, fully responsive e-commerce storefront built with **Next.js 15**, **Zustand**, and **Tailwind CSS**. This project features dynamic product filtering, a custom-built cart system, and a highly polished user interface optimized for all devices.

## ✨ Key Features

- **Premium UI/UX:** Modern "Apple-style" light theme with glassmorphism effects and smooth hover animations.
- **Dynamic Filtering:** Real-time product filtering by **Category**, **Search Query**, and **Price Range** using URL `searchParams`.
- **State Management:** Global state handled by **Zustand** with **Persistent Storage** (cart data remains after page refresh).
- **Custom Success Modal:** A polished, interactive checkout experience that provides user feedback and clears the cart upon completion.
- **Fully Responsive:** Adaptive Navbar with a dedicated mobile search overlay and responsive product grid.
- **Performance Optimized:** Utilizes Next.js **Server Components** for efficient data handling and **Suspense** for client-side hydration.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)
